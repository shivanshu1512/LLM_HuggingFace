# Chapter 06: The 🤗 Tokenizers Library

Building tokenizers from scratch — BPE, WordPiece, Unigram algorithms, fast tokenizers, and custom corpus training.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_tokenization_algorithms.ipynb` | BPE, WordPiece, and Unigram — how they work |
| 📓 `02_training_a_new_tokenizer.ipynb` | Training a custom tokenizer on your own corpus |
| 📓 `03_fast_tokenizers_offsets.ipynb` | Fast tokenizers, offset mappings for NER |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Understand BPE, WordPiece, and Unigram tokenization
- Train a tokenizer from scratch on custom text
- Use fast tokenizers and offset mappings for NER-style tasks
