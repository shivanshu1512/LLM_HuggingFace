# Chapter 03: Fine-Tuning a Pretrained Model

Adapting pretrained transformer models to custom tasks using the Trainer API and custom PyTorch training loops.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_data_preprocessing.ipynb` | Tokenizing datasets for classification tasks |
| 📓 `02_trainer_API.ipynb` | Training with HuggingFace `Trainer` |
| 📓 `03_custom_training_loop.ipynb` | Full PyTorch training loop from scratch |
| 📓 `04_evaluation_metrics.ipynb` | F1, accuracy, and more with `evaluate` |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Preprocess datasets for fine-tuning classification tasks
- Use the HuggingFace `Trainer` API
- Write custom PyTorch training loops
- Evaluate models with standard NLP metrics
