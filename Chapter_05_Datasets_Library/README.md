# Chapter 05: The 🤗 Datasets Library

Efficiently working with large NLP datasets — loading from Hub, preprocessing with map/filter, streaming, and uploading custom datasets.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_loading_datasets.ipynb` | Loading from Hub, CSV, JSON, and local files |
| 📓 `02_slicing_and_dicing.ipynb` | map, filter, shuffle, select, rename |
| 📓 `03_big_data_streaming.ipynb` | Streaming huge datasets without RAM overflow |
| 📓 `04_creating_your_dataset.ipynb` | Building and pushing custom datasets to Hub |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Load datasets from HuggingFace Hub and local files
- Apply map, filter, shuffle, and select operations
- Stream huge datasets without memory overflow
- Create and publish custom datasets
