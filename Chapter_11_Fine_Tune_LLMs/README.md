# Chapter 11: Fine-Tune Large Language Models

Efficient fine-tuning of large models using LoRA, QLoRA, PEFT, instruction tuning, RLHF, and DPO.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_peft_lora_intro.ipynb` | Introduction to PEFT and LoRA adapters |
| 📓 `02_qlora_quantized_finetuning.ipynb` | 4-bit quantization + LoRA with bitsandbytes |
| 📓 `03_instruction_tuning.ipynb` | Fine-tuning on instruction-response pairs |
| 📓 `04_rlhf_dpo.ipynb` | RLHF and Direct Preference Optimization |
| 📓 `05_full_finetuning_pipeline.ipynb` | End-to-end pipeline: prep → train → evaluate → push |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Apply LoRA and QLoRA for memory-efficient fine-tuning
- Use the PEFT library for adapter-based methods
- Implement instruction tuning and chat model training
- Understand RLHF and DPO alignment techniques
