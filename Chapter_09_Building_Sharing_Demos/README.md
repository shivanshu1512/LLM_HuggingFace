# Chapter 09: Building and Sharing Demos

Creating interactive ML demos with Gradio and deploying them to HuggingFace Spaces for free public access.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_gradio_intro.ipynb` | Building your first Gradio interface |
| 📓 `02_gradio_blocks.ipynb` | Advanced layouts with Gradio Blocks |
| 📓 `03_deploying_to_spaces.ipynb` | Publishing demos to HuggingFace Spaces |
| 📓 `04_embedding_demos.ipynb` | Embedding Spaces demos in portfolios |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Build interactive NLP demos with Gradio
- Deploy models to HuggingFace Spaces
- Use Gradio Blocks for advanced UI layouts
- Share and embed demos in portfolios
