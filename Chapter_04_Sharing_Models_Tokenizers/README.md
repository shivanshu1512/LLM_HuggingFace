# Chapter 04: Sharing Models and Tokenizers

Publishing models and tokenizers to the HuggingFace Hub — version control for ML models, Model Cards, and community sharing.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_using_pretrained_models.ipynb` | Finding and loading models from the Hub |
| 📓 `02_push_to_hub.ipynb` | Uploading models with `push_to_hub()` |
| 📓 `03_model_cards.ipynb` | Writing professional Model Cards |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Upload models and tokenizers to HuggingFace Hub
- Write informative Model Cards
- Use `push_to_hub()` and `huggingface_hub` utilities
