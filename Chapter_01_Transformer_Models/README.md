# Chapter 01: Transformer Models

Understanding the architecture behind modern NLP — self-attention, encoder/decoder models (BERT, GPT, T5), and transfer learning fundamentals.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_what_are_transformers.ipynb` | Overview of transformer models and their capabilities |
| 📓 `02_self_attention_explained.ipynb` | Deep dive into the self-attention mechanism |
| 📓 `03_encoder_decoder_models.ipynb` | BERT (encoder), GPT (decoder), T5 (encoder-decoder) |
| 📓 `04_pipeline_API_quickstart.ipynb` | Using HuggingFace `pipeline()` for zero-shot NLP |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Explain the Transformer architecture and self-attention mechanism
- Distinguish between encoder, decoder, and encoder-decoder models
- Use `pipeline()` for quick NLP inference
