# Chapter 07: Classical NLP Tasks

End-to-end notebooks for NER, masked LM pre-training, translation, summarization, and causal language modeling.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_token_classification_ner.ipynb` | Fine-tuning for Named Entity Recognition |
| 📓 `02_masked_language_modeling.ipynb` | MLM pre-training on custom corpora |
| 📓 `03_translation_seq2seq.ipynb` | Neural machine translation with MarianMT/mBART |
| 📓 `04_summarization.ipynb` | Abstractive summarization with BART/T5 |
| 📓 `05_causal_lm_generation.ipynb` | Text generation with GPT-2 and beyond |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Fine-tune models for NER with token classification
- Apply masked LM pre-training
- Build translation and summarization pipelines
- Generate text with causal language models
