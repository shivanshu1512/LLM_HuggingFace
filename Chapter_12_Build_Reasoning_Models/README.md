# Chapter 12: Build Reasoning Models ✨ NEW

Building reasoning-capable AI systems with Chain-of-Thought prompting, ReAct agents, tool use, and HuggingFace smolagents.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_chain_of_thought_prompting.ipynb` | Few-shot and zero-shot Chain-of-Thought |
| 📓 `02_react_framework.ipynb` | ReAct: Reasoning + Acting agent loop |
| 📓 `03_tool_use_function_calling.ipynb` | Giving LLMs tools: search, calculator, code |
| 📓 `04_smolagents_intro.ipynb` | Building agents with HuggingFace smolagents |
| 📓 `05_building_autonomous_agents.ipynb` | End-to-end autonomous AI agent project |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Apply Chain-of-Thought prompting for complex reasoning
- Build ReAct agents that reason and act in loops
- Enable tool use and function calling in LLMs
- Create autonomous AI agents with `smolagents`
