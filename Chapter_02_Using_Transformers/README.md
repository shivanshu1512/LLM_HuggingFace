# Chapter 02: Using 🤗 Transformers

Hands-on usage of the HuggingFace `transformers` library — AutoModel, AutoTokenizer, tokenization internals, and running inference.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_behind_the_pipeline.ipynb` | What happens inside a `pipeline()` call |
| 📓 `02_tokenizer_deep_dive.ipynb` | Input IDs, attention masks, token type IDs |
| 📓 `03_models_and_configs.ipynb` | AutoModel, AutoConfig — loading any architecture |
| 📓 `04_multiple_sequences.ipynb` | Batching, padding, and truncation |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Use `AutoModel` and `AutoTokenizer` for any pretrained model
- Understand tokenization: input IDs, attention masks, special tokens
- Run multi-sequence batches through transformer models
