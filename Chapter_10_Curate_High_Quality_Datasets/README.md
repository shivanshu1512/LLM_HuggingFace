# Chapter 10: Curate High-Quality Datasets

Dataset curation best practices — collection, deduplication, quality filtering, annotation with Argilla, and ethical considerations.

---

## 📓 Notebooks in this Chapter

| Notebook | Description |
|----------|-------------|
| 📓 `01_data_collection_strategies.ipynb` | Web scraping and dataset collection |
| 📓 `02_deduplication_filtering.ipynb` | MinHash deduplication and quality filters |
| 📓 `03_annotation_with_argilla.ipynb` | Human annotation workflows with Argilla |
| 📓 `04_dataset_ethics.ipynb` | Bias, representation, and responsible AI |

---

## 🎯 Learning Objectives

By the end of this chapter, you will be able to:

- Design data collection pipelines for NLP
- Apply deduplication and quality filtering
- Use Argilla for human annotation
- Evaluate dataset quality and address ethical concerns
